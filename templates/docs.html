Bot Documentation

Command Reference

/start

Starts the bot and shows the welcome message.

/start

/help

Displays a list of available commands and their descriptions.

/help

/info

Returns current information about cryptocurrency pools on Raydium, including APR, volume, and liquidity data.

/info

/simulate

Simulates investment returns based on the current pool data, including projections and impermanent loss risk.

/simulate 1000
Replace 1000 with your investment amount.

/status

Shows the current status of the bot, including uptime, user count, and API status.

/status

/subscribe

Subscribes you to receive daily updates about the best performing pools.

/subscribe

/unsubscribe

Unsubscribes you from daily updates.

/unsubscribe

/ask

Asks an AI-powered question about cryptocurrency, pools, or investment strategies.

/ask What is impermanent loss?
Replace the question with your own query.

Natural Language Queries

You can also send direct messages to the bot with questions or keywords, and it will try to understand and provide a relevant response using AI. For example:

  • "What pools have the highest APR?"
  • "Show me stable pools with low risk"
  • "How does impermanent loss affect my returns?"
  • "What's the average APR for USDC pools?"

Technical Information

This bot uses the following technologies and APIs:

  • python-telegram-bot - For the Telegram bot interface
  • Raydium API - For real-time cryptocurrency pool data
  • DeepSeek AI - For natural language processing and question answering

Data is cached to minimize API calls and provide faster responses. Pool data is cached for 5 minutes, and simulation results are cached for 10 minutes.
